# 硬件设计说明文档

**项目名称：** 智能交通灯控制系统

**晶振：** 11.0592 MHz

## 二、最小系统电路

### 2.1 晶振电路
| 元件 | 参数 | 连接 |
|------|------|------|
| 晶振 Y1 | 11.0592 MHz | XTAL1(19) ↔ XTAL2(18) |
| C1 | 30 pF | XTAL1 ↔ GND |
| C2 | 30 pF | XTAL2 ↔ GND |

### 2.2 复位电路
| 元件 | 参数 | 连接 |
|------|------|------|
| R1 | 10 kΩ | VCC → RST(9) |
| C3 | 10 μF 电解 | RST(9) → GND |
| S1 | 按键 | 并联在 C3 两端 |

### 2.3 EA 引脚
- EA/VPP(31) 直接接 VCC（使用内部程序存储器）

### 2.4 电源
- VCC(40) → +5V
- GND(20) → GND

## 五、蜂鸣器驱动电路

```
P2.7 ───[500Ω]───┐
                  │
               ┌──┴──┐
               │ NPN  │
               └──┬──┘
                  │
               ┌──┴──┐
               │BUZZER│
               └──┬──┘
                  │
                  VCC
```

- P2.7 = 1 → NPN 导通 → 蜂鸣器两端产生电压差 → 鸣响
- P2.7 = 0 → NPN 截止 → 蜂鸣器无电流 → 静音

## 六、Proteus 仿真注意事项

1. AT89C52 的 Program File 必须指向 `Objects\Projectr.hex`
2. Clock Frequency 设为 `11.0592MHz`
3. 共阴数码管选 `7SEG-MPX2-CC`，位选低电平导通
4. OLED 选 `SSD1306`，I2C 地址 0x78
5. 蜂鸣器选 `BUZZER` (DC/Active) 型号
6. NPN 三极管选通用 `NPN`，不要选 2N2222（Proteus 中驱动蜂鸣器可能不响）
