## 一、系统架构

P-A-1# 温湿度检测系统由**两块目标板**组成，通过排针排母互联：

| 属性 | 目标板1（底板） | 目标板2（扩展板） |
|:---|:---|:---|
| 核心芯片 | STC89C54RD+ (DIP40) | 8255A (DIP40) |
| 晶振 | 12MHz | 无独立晶振 |
| ADC | ADC0809 (DIP28) | 无 |
| 地址锁存 | 74HC373 (DIP20) | 无（P0.0/P0.1直连8255 A0/A1） |
| 反相驱动 | 74HC240 (DIP20) | ULN2003A (DIP16) |
| 译码 | 74HC02 (DIP14) 或非门 | 无 |
| 数码管 | 8位共阳 | 6位共阴 FJ3661AH |
| 传感器 | 模拟温度DW1 + 模拟湿度DWQ | DHT11 (P3.3) |
| 按键 | S0-S7 (8个) + KRS1复位 | SK0-SK7 (8个) |
| LED | LED0-LED7 (8个) | D1-D8 (8个，8255 PA口经74HC240驱动) |
| 串口 | CH341T USB + MAX485 RS485 + S9跳线 | 无 |

### 2.6 74HC02 或非门译码 (U8, DIP14)

| 门 | 输入1 | 输入2 | 输出 | 连接 |
|:---|:---|:---|:---|:---|
| U8A | /WR(P3.6, 2脚) | A7(74HC373 Q7, 3脚) | 1Y(1脚) | → ADC0809 ALE(22脚)+START(6脚) |
| U8B | /RD(P3.7, 5脚) | A7(74HC373 Q7, 6脚) | 2Y(4脚) | → ADC0809 OE(9脚) |

**译码逻辑**：
- ADC写：`ALE/START = !(/WR + A7)` = 当/WR=0且A7=1时，输出高电平启动ADC
- ADC读：`OE = !(/RD + A7)` = 当/RD=0且A7=1时，输出高电平使能数据输出

### 2.7 数码管（8位共阳）

- **类型**：共阳极，阳极经RP5(1K×8)上拉至VCC
- **段选**：MCU P0.0-P0.7 → 74HC240反相 → a-g,dp段（低电平点亮）
- **位选**：MCU P2.0-P2.7 → SK0-SK7（低电平选中）
- **动态扫描**：P2口轮流输出低电平选中位，P0口同步输出段码

### 2.8 LED指示灯

- **LED0-LED7**：8个，阳极经RP5(1K×8)→VCC，阴极→74HC240输出→P0口控制
- **LED8**：独立电源指示灯，阳极→VCCU，经R14(510Ω)和光耦→GND
- **点亮方式**：P0输出高电平→74HC240输出低电平→LED亮

## 五、两板联合工作模式

P-A-1# 温湿度检测流程中：

- **底板独立工作**：ADC0809读取IN0(DW1)模拟温度 + IN1(DWQ)模拟湿度，数码管显示，LED指示
- **扩展板独立工作**：DHT11读取数字温湿度，8255驱动6位数码管显示，16个按键输入
- **联合工作**：MCU通过P0总线同时访问底板ADC0809和扩展板8255A，通过不同的片选地址(ADC:0x4000, 8255:0x0480)区分
